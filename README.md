# 🎪 Quitanda Digital | Lista de Compras

Um aplicativo web interativo para gerenciar listas de compras, desenvolvido com um visual nostálgico de quitanda aliado a tendências modernas de design de interface como o **Glassmorphism**.

Este projeto foi construído como resolução de um desafio prático da **Rocketseat**, focado em manipulação do DOM, eventos no JavaScript e estilização avançada com CSS.

> 🔗 **[Clique aqui para acessar o projeto online](https://cecilialopess.github.io/Lista-de-compras-quitanda/)**

---

## ✨ Funcionalidades

* **Adicionar itens:** Adicione rapidamente produtos à sua lista.
* **Marcar como concluído:** Checkboxes personalizados para gerenciar o que já foi comprado.
* **Remover itens:** Exclusão de itens com feedback visual.
* **Alerta de Remoção (Toast):** Mensagem de confirmação que aparece ao deletar um item e desaparece automaticamente após 3 segundos.

---

## 🛠️ Tecnologias e Técnicas Utilizadas

* **HTML5:** Estrutura semântica para melhor acessibilidade.
* **CSS3 Avançado:**
    * **Glassmorphism:** Fundo de vidro fosco utilizando `backdrop-filter: blur()`.
    * **CSS Art:** Tenda de quitanda criada **100% com código**, usando `repeating-linear-gradient` para as listras e `radial-gradient` para o acabamento arredondado das bordas.
    * **Degradê Dinâmico:** Fundo com `linear-gradient` para destacar o efeito de transparência.
* **JavaScript (Vanilla):**
    * Manipulação dinâmica de elementos do DOM.
    * Escuta e delegação de eventos.
    * Lógica de temporização para alertas de sistema.
* **Phosphor Icons:** Ícones modernos e minimalistas.
